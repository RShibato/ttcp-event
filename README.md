# academist × 日本総研 イベントLP

## ファイル構成

```
/
├── index.html        ← メインHTML
├── ogp.png           ← OGP画像（1200×630px）
└── images/
    ├── higashi.jpg   ← 東 博暢 さんの写真
    ├── hayano.jpg    ← 早野 元詞 さんの写真
    └── chiba.jpg     ← 千葉 のどか さんの写真
```

## 写真の差し替え方

`images/` フォルダ内のプレースホルダー画像を、実際の写真に差し替えてください。

- 推奨サイズ：**400×400px 以上の正方形**（自動でトリミングされます）
- 形式：JPEG または PNG
- ファイル名は上記のとおり正確に（大文字・小文字も一致させること）
